# Ember Glow Desk

Portfólio full-stack em **TanStack Start** (React + Vite + Nitro).

## Stack

- React 19 + TanStack Router/Start (SSR)
- Tailwind CSS v4
- shadcn/ui (Radix UI)
- Three.js / React Three Fiber (cena 3D)
- Nitro (build do servidor)

## Convenções de rotas

Veja `src/routes/README.md` — roteamento baseado em arquivos do TanStack Router.


## Scripts

- `npm run dev` — servidor de desenvolvimento
- `npm run build` — build de produção
- `npm run preview` — pré-visualiza o build de produção
- `npm run lint` — checa lint
- `npm run format` — formata o código com Prettier
